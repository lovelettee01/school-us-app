# Phase 7 - 시간표 탭 와이어프레임

작성일: 2026-03-26
목표: 상세 페이지 탭 3(시간표)의 선택 UI 및 결과 레이아웃을 확정한다.

## 1) 데스크톱 와이어프레임(텍스트)

```text
+----------------------------------------------------------------------------------+
| [탭] 학교정보/위치 | [탭] 급식 | [탭] 시간표(활성)                                 |
+----------------------------------------------------------------------------------+
| 학교급: [자동표시]  학년: [1▼]  학급: [1▼]  날짜/주간: [2026-03-26▼] [조회]      |
| 안내: 학년/학급을 선택한 뒤 조회하면 해당 시간표를 표시합니다.                    |
+----------------------------------------------------------------------------------+
| [요약] 학교명 / 학년-학급 / 기준일(또는 주간) / 총 교시 수                         |
+----------------------------------------------------------------------------------+
| 교시 | 월 | 화 | 수 | 목 | 금                                                     |
| 1교시| 국어 | 수학 | 영어 | 과학 | 체육                                           |
| 2교시| ...                                                                       |
| ...                                                                              |
+----------------------------------------------------------------------------------+
| 상태 영역: 미선택 안내 / 로딩 / 빈값 / 오류 + 재시도                              |
+----------------------------------------------------------------------------------+
```

## 2) 모바일 와이어프레임(텍스트)

```text
+--------------------------------------+
| [탭] 정보 | 급식 | [시간표*]          |
+--------------------------------------+
| 학교급(자동)                          |
| [학년 선택] [학급 선택]               |
| [기준일/주간 선택] [조회]             |
+--------------------------------------+
| 요약: 학교명 / 학년-학급 / 기준일     |
+--------------------------------------+
| [교시별 아코디언 or 가로 스크롤 표]   |
| 1교시: 월 국어 / 화 수학 ...          |
+--------------------------------------+
| 미선택/로딩/빈값/오류 상태            |
+--------------------------------------+
```

## 3) 선택 컨트롤 규칙

- 학교급(`elementary|middle|high`)은 상세 공통 데이터에서 자동 결정
- 학년/학급은 기본값 미선택 상태로 시작
- 조회 트리거는 `조회 버튼` 기본
- 옵션: 마지막 선택값 재진입 시 복원(동일 schoolKey)

## 4) 조회 기준 규칙

- 기본 기준: 오늘 날짜 포함 주간
- 사용자가 일자 단일 조회 또는 주간 조회 선택 가능
- 주간 조회 시 월~금 중심으로 렌더(주말은 빈칸)

## 5) 결과 표시 방식

- 데스크톱: 표 형태 우선
- 모바일: 교시 아코디언 또는 가로 스크롤 표
- 교시 정렬: 숫자 오름차순(1교시 -> n교시)

## 6) 상태 UX

- 미선택: "학년과 학급을 선택해 주세요"
- 로딩: 표 스켈레톤
- 빈값: "선택한 조건의 시간표가 없습니다"
- 오류: "시간표를 불러오지 못했습니다" + 재시도

## 7) 접근성/반응형 규칙

- 선택 컨트롤 라벨 명시(`학년`, `학급`, `기준일`)
- 키보드로 선택/조회 가능
- 표 헤더와 셀의 의미 연결(스크린리더용)
- 모바일 최소 글자 크기/터치 영역 유지
