# Phase 0 - 상태 전이표

작성일: 2026-03-26
범위: 검색 화면, 상세 화면, 탭별 데이터 조회의 공통 상태를 정의한다.

## 1) 검색 화면 상태

| 상태 | 진입 조건 | 사용자 표시 | 다음 전이 |
|---|---|---|---|
| 초기(Idle) | 첫 진입 | 빈 입력 폼, 안내 문구 | 조회 클릭 -> 로딩 |
| 로딩(Loading) | 유효한 조건으로 조회 시작 | 스피너/스켈레톤, 버튼 비활성 | 성공 -> 성공, 결과 0건 -> 빈값, 실패 -> 오류 |
| 성공(Success) | 결과 1건 이상 | 학교 리스트 표시 | 재조회 -> 로딩, 항목 클릭 -> 상세 페이지 이동 |
| 빈값(Empty) | 응답 성공 + 결과 0건 | "검색 결과 없음" | 조건 수정 후 재조회 -> 로딩 |
| 오류(Error) | 네트워크/API/파라미터 오류 | 오류 메시지 + 재시도 버튼 | 재시도 -> 로딩 |

## 2) 상세 화면 공통 상태

| 상태 | 진입 조건 | 사용자 표시 | 다음 전이 |
|---|---|---|---|
| 초기(Idle) | 상세 진입 직후 | 기본 레이아웃, 대표정보 자리표시자 | 대표정보 조회 -> 로딩 |
| 로딩(Loading) | 상세 데이터 조회 시작 | 대표정보/탭 콘텐츠 스켈레톤 | 성공 -> 성공, 실패 -> 오류 |
| 성공(Success) | 대표정보 조회 성공 | 학교 대표정보 + 탭 노출 | 탭 변경 -> 탭별 로딩/성공/오류 |
| 오류(Error) | 학교 식별자 오류/API 실패 | 오류 안내 + 이전으로/재시도 | 재시도 -> 로딩 |

## 3) 탭별 상태(학교정보/지도, 급식, 시간표 공통)

| 상태 | 진입 조건 | 사용자 표시 | 다음 전이 |
|---|---|---|---|
| 초기(Idle) | 탭 첫 오픈 전 | 탭 안내 문구 | 탭 진입/조회 조건 충족 -> 로딩 |
| 로딩(Loading) | 탭 데이터 조회 중 | 탭 전용 스켈레톤 | 성공 -> 성공, 빈값 -> 빈값, 실패 -> 오류 |
| 성공(Success) | 데이터 조회 성공 | 탭 콘텐츠 렌더링 | 조건 변경/새조회 -> 로딩 |
| 빈값(Empty) | 조회 성공 + 데이터 없음 | "조회 데이터 없음" | 날짜/학년/학급 변경 -> 로딩 |
| 오류(Error) | API 실패/매핑 실패 | 오류 메시지 + 재시도 | 재시도 -> 로딩 |

## 4) 다크모드 상태

| 상태 | 진입 조건 | 사용자 표시 | 다음 전이 |
|---|---|---|---|
| 시스템기본(System) | 저장값 없음 | OS 설정에 따라 테마 적용 | 토글 클릭 -> 라이트 또는 다크 |
| 라이트(Light) | 사용자 선택 | 라이트 색상 토큰 적용 | 토글 클릭 -> 다크 |
| 다크(Dark) | 사용자 선택 | 다크 색상 토큰 적용 | 토글 클릭 -> 라이트 |

## 5) 글로벌 예외 상태 정책

- 입력 오류: 필드 하단 인라인 메시지로 즉시 안내한다.
- API 제한/서버 오류: 상단 배너 또는 섹션 오류 박스로 안내한다.
- 재시도 가능 오류: 명시적인 `재시도` 액션을 제공한다.
