# 상세 화면(`/school/[schoolKey]`) 상세 재구축 플랜

## 1. 화면 목표
- 학교 대표 정보, 지도/거리, 급식, 시간표를 탭 기반으로 조회한다.

## 2. 라우팅/초기 진입 규칙
- path param: `schoolKey`
- `schoolKey` 파싱 실패 시 오류 상태
- query param `tab` 지원
  - 허용값: `info | meal | timetable`
  - 미허용값은 `info`로 정규화

## 3. 공통 상단 구성
- 상단 헤더
  - 페이지명 + 학교명
  - 테마 토글
- `SchoolHeader`
  - 학교명/교육청/학교급
  - 도로명 주소(지번 보조)
  - 전화번호(옵션)
  - 홈페이지 링크(옵션)
  - 즐겨찾기 토글
  - 홈 이동 버튼
- `Tabs`
  - 3탭 + 아이콘
  - role/tablist 접근성
  - 좌우 키 이동

## 4. 탭1: 학교정보/지도/거리
### 구성
- 기본정보 아코디언
- 지도 패널
- 거리/길찾기 패널

### 동작 규칙
- 지도 표시 우선순위
  - 1순위: 상세 좌표(lat/lng)
  - 2순위: 도로명 주소 지오코딩
  - 실패: error/empty 상태 메시지
- 거리 계산
  - 현재 위치 권한 요청
  - 직선거리 표시(m 또는 km)
  - 도보/자전거/차량 예상 시간 표시
- 길찾기
  - 새 탭 열기
  - 링크 복사

### 예외 처리
- SDK 로드 실패
- 주소 없음
- 위치 권한 거부
- 팝업 차단

## 5. 탭2: 급식
### 구성
- 기간 선택(시작일/종료일)
- 조회 버튼
- 알레르기 가이드 툴팁
- 급식 결과 아코디언
- 영양/원산지/열량 카드

### 동작 규칙
- 초기 진입 시 오늘 날짜 기준 자동 조회
- 결과는 날짜 오름차순
- 첫 항목 기본 펼침
- 메뉴 문자열에서 알레르기 코드 파싱
- TODAY 배지 표시

### 예외 처리
- 날짜 형식 오류
- 시작일 > 종료일
- 빈급식(주말/휴일 포함)
- API 실패 재시도

## 6. 탭3: 시간표
### 구성
- 주간/일자 토글
- 조회기준일
- 학년/학급 선택
- 조회 버튼
- 표 형태 결과(교시 x 날짜)

### 동작 규칙
- 학교급 판별
  - 초등: 1~6학년
  - 중/고: 1~3학년
- 학년/학급 미선택 시 조회 제한
- 주간 모드: 기준일이 포함된 월~금 범위 조회
- 일자 모드: 기준일 단일 조회

### 예외 처리
- 학교급 판별 실패
- 빈시간표
- API 실패 재시도

## 7. 데이터/상태 설계
- `useSchoolDetail`
  - 상태: `loading | success | empty | error`
  - 캐시 TTL: 10분
- `useMeals`, `useTimetable`
  - 상태: `idle | loading | success | empty | error`
  - 캐시 TTL: 5분
- 탭 전환 시 URL 동기화로 직접 링크 가능 상태 유지

## 8. 테스트 포인트
- `schoolKey` 파싱 테스트
- 탭 정규화 테스트
- 지도 URL 생성/거리 계산 테스트
- 급식 매퍼/시간표 매퍼 테스트
- 상세 통합 시나리오
  - 탭 이동
  - 지도/거리 계산
  - 급식 조회
  - 시간표 조회
